Experiment 2 - Living Impressions

Description

The purpose of this project is to recreate the following image that is responsive to time and the user.

src: https://sierramountainpasses.com/about-the-sierra-nevada-passes/central-sierra/

Technical

This project was created with p5js. The continous mountains are created using noise. The trees are created using multiple triangles.
Everything else is made up of rectangles. Movement is achieved with every update to draw(), mainly with the use of millis().
The sun sets it's position to the mouse position. Random() is used for parts such as the position of the trees.

Reflection

I am the only one who worked on this project. It was interesting to slowly build up the scene layer by layer.
The most difficult part was getting the movement to loop correctly. I think this experiment was a success,
especially the mountain and the parallax effect.
