Experiment 3 - Alternate Worlds - Overworld

Description

Using noise to create round natural looking biomes. The player can move a character with WASD or the arrow keys. There are three different biomes, grass, dirt, and stone. Trees can only grow on the grass biome.
Codes: 'g' == grass, 'd' == dirt, 's' == stone, 't' == tree
Life: There is a bird that moves in a random direction. The player character has an idle animation.
using tiles from [Micro Tileset - Overworld & Dungeon] by [thkaspar]

Technical

An array keeps track of the location of tiles. A script then takes that array and fills in a grid with corresponding tiles from a tilesheet. The array is generated using perlin noise, and the biome is determined by the value of the noise at specific coordinates. Tiles check their neighbors to determine how to make transitions.

Reflection

I worked on this project alone. The difficult part of this was figuring out how to map tiles to the grid and figuring out how to make them transition. Having the player character being able to move around in the map is a success in my opinion.
