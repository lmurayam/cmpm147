Experiment 4 - Infinite Worlds - A

Description

Infinity
As a fan of strategy games such as those in the Civilization series, I created an infinitely extending hexagonal terrain.
Variation in Space
There are three distinct terrain types in my terrain, those being water, sand, and grass. Viewers can see that the sand represents a beach for the transition between water and land. The grass tiles can have trees on them.
Variations in Key
Changing the key changes the layout of the world. The key changes the noise seed which determines where the biomes are located and the inital elevation of tiles. The key is also used to determine where trees can spawn and the timing of the waves.
Signs of Life
Clicking a tile causes it's elevation to rise up. Water tiles move up and down as time passes to emulate waves.

Technical

The world key is hashed using xxhash and converted into a number. That number is used as a seed for both random numbers and noise. This value is also used for hashing other values in the world which are used to create replicatable random distributions. The hexagonal tiles are made by taking points on a circle that are rotated 1/6th degrees. Every other row is offset so the hexagons can tile the plane. The fake 3D effect is made by redrawing the hexagon above the original by a certain amount, then drawing lines down which connect to the original. The program overdraws the area so when the camera moves new tiles don't just pop in. Noise is used to distribute the different biomes and hashing the tile location with the world hash is used to determine if there should be a tree. In other versions of this project, this may be slighly different. A dictionary is used to keep track of if a tile is clicked.
