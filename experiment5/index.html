Description

Given a specific image, this program outputs a version that is rendered using rectangles.

Technical

1. This program takes an image.
2. The canvas is resized to the image, if the image is too large based on a slider then the image is scaled down.
3. The program then creates x amount of random rectangles across the canvas from three different scales.
4. The color of the rectangles are based on the average color the area takes up on the original image.
5. The program then iterates through the pixels of the canvas comparing it with the original image and gives it a score.
6. The highest scores are saved.
7. The rectangles are then mutated based on the mutation rate, the mutation result in small movement in the x and y and small changes in the rectangle size.
8. Repeat from step 4.

Reflection

This was a fun project. I've seen someone do something familiar to this for geometry dash. There is also a more complex version of this in geometrize. I worked on this project by myself. The most annoying part was finding out that resizing the canvas to a width and height that are not whole numbers would break my coloring function. But on the flip side, when it finally started working everything pretty much came together.
